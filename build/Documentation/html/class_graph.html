Detailed Description

template<class T, class A>
class Graph< T, A >

The Graph-class provides a "bucket" for GraphNodes, which cann be added to the vector-list ot the Graph. A Graph represents one closed path-system on which a AI-Unit can run. For example one Graph could represent the search for food or the area, which the AI should protect.
