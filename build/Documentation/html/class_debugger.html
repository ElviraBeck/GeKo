Detailed Description

To use the debugger in a main-method you have to initialize the debugger-object before creating the window. The debug-callback has to be set after calling "glewInit();". You can define your own error messages using void DebugMessageInsert(enum source, enum type, uint id, enum severity, sizei length, const char* buf). For further information have a look at https://www.opengl.org/registry/specs/KHR/debug.txt
