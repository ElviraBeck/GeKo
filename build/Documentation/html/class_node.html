void Node::addRotation ( float  angle,
glm::vec3  axis 
)

A rotation will be added to the Modelmatrix.

The user tells the method how big the angle should be and around which axis we rotate. Then the method calls the rotate(...) Method of opengl and updates the modelmatrix itself! This method should only be used once, because every time it is called, it overwrites the rotation from before!

void Node::addScale ( float  x,
float  y,
float  z 
)

A scale will be added to the Modelmatrix.

The user tells the method in which directions (x, y and/or z) the object should be scaled. Then the method calls the scale(...) Method of opengl and updates the modelmatrix. This method overwrites all scalings from before! Furthermore if the node has an bounding-sphere, the radius of it will be updated as well!

void Node::addTexture ( Texture *  texture)
void Node::addTranslation ( float  x,
float  y,
float  z 
)

A translation will be added to the Modelmatrix.

The user tells the method on which position an object should be! Then the method calls the translate(...) Method of opengl and updates the modelmatrix! This method overwrites all translations from before and changes the modelmatrix completely to its new position value! Furthermore if the node has an bounding-sphere, the center of it will be updated as well!
