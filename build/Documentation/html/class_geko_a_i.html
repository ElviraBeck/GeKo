Detailed Description

This class is a first generalisation of the Player class. A Geko has a view direction and a movement vector, so it can be moved over the playground. The Geko-class also has its own movement Methods and each frame it will be updatet, so the player can move

A update Method to update the position.

If the movement vector has changed, the new position will be calculated. The model matrix of the geometry will be updated as well. If the view rotated, this method will update the model matrix additionally. The method will be called in each frame!
