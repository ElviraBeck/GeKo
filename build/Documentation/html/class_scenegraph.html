Detailed Description

Scenegraph contains Node.

Every scenegraph is connected with one scene and its name is the same as the scenes name it belongs to. The scenegraph is managing all the objects in the scene which are added to the scenegraph by adding them to the scene. Each scenegraph provides a rootnode from the beginning
