InputMap * InputHandler::getActiveInputMap ( )

The method returns the active InputMap.

Iterates over m_allInputMaps, where all the InputMaps are stored and returns the one, whose member variable m_active is set true. If there is no active InputMap the program exits with a message saying there's no InputMap active If there is more than one active InputMap the program exits with a message saying it's not the only map being active

InputMap * InputHandler::getInputMap ( std::string  name)

The method returns the InputMap instance with the matching name.

If there is any InputMap named as the parameter the according InputMap is returned, otherwise an error message is given.
