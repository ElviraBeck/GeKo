5 /*
6 * @brief Class for OpenGL frame buffer objects (FBOs).
7 *
8 * A FBO has a given width and height. All textures used by it
9 * or created by it have these values for width and height. It
10 * is possible to create a fbo with multiple color buffers, each
11 * with rgba values per pixel. A fbo can use a depth buffer and
12 * a stencil buffer, which are not necesserily needed.
13 *
14 * To create a fbo, the constructor has to be called with the
15 * preferred width and height. Additionally the number of color
16 * textures and the use of depth or stencil buffer can be assigned.
17 * An example creation of a fbo with 1 color buffer and a depth
18 * buffer is:
19 * CVK::FBO fbo( 400, 300, 1, true, true);
20 *
21 * To use a fbo, it has to be bound to OpenGL. If the standard
22 * OpenGL frame buffer should be used, the fbo has to be unbound.
23 * fbo.bind();
24 * ...
25 * fbo.unbind();
26 *
27 * To use the result of the fbo, getters for the colorTextures
28 * and the depth texture exist.
29 * GLuint colorTexture = fbo.getColorTexture(0);
30 * GLuint depthTexture = fbo.getDepthTexture();
31 */
